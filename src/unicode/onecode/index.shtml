UNICODE

One Code

256 characters for the Macintoshes leading innovation,
256 for the Wintel boxes destined for world domination,
256 for the homemade operating systems doomed to die,
256 for the Linux users fighting for free information
in the land of open source where the GPL licenses lie.
One Code to rule them all, One Code to find them,
One Code to bring them all and in the darkness bind them,
in the land of computing where the text encodings lie.

— Rebecca Bettencourt, July 2003, December 2010.
